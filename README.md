# MCSR 全类型种子自动输入工具
专为MCSR Ranked速通练习设计，支持多类型开局种子自动获取与一键输入，替代手动复制粘贴，大幅提升速通练习效率。

## 功能特性
- 自动调用API获取**6种主流开局类型**种子（宝藏/废门/沙漠神殿/村庄/沉船/随机）
- F5一键触发游戏内种子输入全流程，F6安全退出程序
- 纯键盘逐字符模拟输入，彻底避免剪贴板粘贴的输入错误
- 后台预加载下一批种子，保证操作连贯性
- 严格匹配MCSR Ranked排位赛种子规范（completion=720000）

## 环境准备
### 1. 基础要求
已安装Python 3.8及以上版本（推荐3.10+），确保网络可访问MCSR Ranked API地址。

### 2. 安装依赖库
打开命令行执行以下命令，安装工具所需Python库：
```bash
pip install requests pynput pyinstaller
```
#### 依赖库说明
| 库名       | 核心作用                     |
|------------|------------------------------|
| requests   | 发送网络请求，获取API种子数据 |
| pynput     | 监听键盘按键、模拟键盘输入    |
| pyinstaller| 将Python脚本打包为可执行文件  |

## 使用方法
### 方式一：直接运行Python脚本
1. 将工具代码保存为`rankbt.py`文件
2. 命令行进入文件所在目录，执行运行命令：
   ```bash
   python rankbt.py
   ```
3. 控制台提示选择开局类型，输入对应数字（1-6）并回车
4. 控制台提示**游戏界面按F5开始练习，按F6退出程序**即启动成功
5. 切换到Minecraft游戏界面，按对应按键操作：
   - **F5**：自动完成种子输入全流程，同时预加载下一批种子
   - **F6**：立即终止程序，安全退出

### 方式二：打包为可执行文件（推荐）
可将脚本打包为独立exe文件，无需Python环境即可运行：
1. 将`RANKED.png`图标文件与`rankbt.py`放在**同一目录**（无图标可省略`-i RANKED.png`参数）
2. 命令行执行打包命令：
   ```bash
   pyinstaller -F -w -i RANKED.png rankbt.py
   ```
3. 打包完成后，在当前目录的`dist`文件夹中找到`rankbt.exe`
4. 双击`rankbt.exe`，按控制台提示选择开局类型后，即可正常使用

## API 相关说明
### 核心调用API
工具调用MCSR Ranked v2版API，根据选择的开局类型获取对应随机种子：
```
http://43.143.231.104:8001/api/v2/seed?overworld={类型标识}&completion=720000
```
### API 参数说明
| 参数          | 取值               | 说明                                                                 |
|---------------|--------------------|----------------------------------------------------------------------|
| overworld     | 对应开局类型标识   | 指定主世界种子类型，为MCSR Ranked支持的合法类型                      |
| completion    | 720000             | 限定为排位模式种子，且原比赛完成时间≤720000毫秒（12分钟）|

## 核心操作流程
按下F5后，程序将自动执行以下操作，无需手动干预：
1. 模拟键盘按键完成游戏内种子输入界面的焦点定位
2. 依次输入**主世界种子** → 切换输入框 → 输入**下界种子** → 切换输入框 → 再次输入**主世界种子**
3. 模拟按键确认种子输入，完成界面提交
4. 后台自动调用API获取下一批对应类型的种子，为下一次F5操作做准备
5. 控制台实时打印当前使用的种子信息，方便核对

## 注意事项
1. 运行程序前，需将Minecraft窗口置于**可操作状态**，确保为当前活动窗口
2. 种子输入过程中有微小字符延迟（0.01秒/字符），为保证输入准确，**请勿在执行过程中手动操作键盘/鼠标**
3. 若程序无响应，优先按F6退出
4. 打包可执行文件时，确保文件名无中文/特殊字符，建议以管理员身份运行命令行
5. 工具仅用于MCSR Ranked速通练习，请勿用于违规游戏场景
6. 若出现API连接失败，检查网络连接是否正常，确认API地址可正常访问

## 常见问题排查
### Q1：按下F5无反应？
A1：① 确认Minecraft窗口为当前活动窗口；② 检查程序是否正常运行（控制台无报错）；③ 关闭其他占用键盘的软件（如输入法、按键精灵等）。

### Q2：种子输入出现缺字/错字？
A2：① 确保游戏内输入框为**英文输入法状态**；② 不要在程序执行时手动操作键盘，等待输入流程完成。

### Q3：打包exe时提示报错？
A3：① 升级pyinstaller到最新版本：`pip install --upgrade pyinstaller`；② 确保文件路径无中文/特殊字符；③ 以管理员身份运行命令行重新打包。

### Q4：API获取种子失败？
A4：① 检查网络连接，能否正常访问API地址
